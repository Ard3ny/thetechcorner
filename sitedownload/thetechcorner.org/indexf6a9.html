First, we need to clarify what Raspberi PI is:

The Raspberry Pi is a low cost, credit-card sized computer that plugs into a computer monitor or TV, and uses a standard keyboard and mouse. It is a capable little device that enables people of all ages to explore computing, and to learn how to program in languages like Scratch and Python. It’s capable of doing everything you’d expect a desktop computer to do, from browsing the internet and playing high-definition video, to making spreadsheets, word-processing, and playing games.

So bassically, the Raspberry Pi is just a small computer that you can program to do pretty much anything with the right sensors and code.

The instructions that you will find here will guide you threw whole process of installation and will concist of the following:

  1. All the necessary equipment you will need
  2. How to download and write the operating system onto SD card
  3. Installation process with a monitor
  4. Installation process without a monitor

All the necessary equipment you will need

  1. Raspberry PI (for this guide I will be using Raspberry PI 4 B, but you can use any model)
  2. Power supply for model you are using (you will need stronger power supply with newer model)
  3. SD Card or USB Flash Drive
  4. Network or HDMI cable depending if you want to use monitor or not (for model 4B you will need micro HDMI)
  5. PC or laptop for downloading the OS and writing the image one SD card / USB

How to download and write the operating system onto SD card

  1. First you will need to download some sort of operating system. There are probabaly at minimum around 20 OS’s you can run on Raspberry PI. You can even run the Windows ARM version on your PI. But then your full small programmable computer that can do pretty much anything kinda loses it’s purpose, right? Therefore, for this example we will run Raspbian, because it’s the easiest one to run and is probably the most compatible one as it was made exactly for this reason. But you can check out all the other possible OS’s here.

You can download three versions of Rasbian. Lite version, OS with desktop GUI version and OS with desktop GUI and all the other recommended software version.

We will continue with the Raspberry PI OS with desktop and recommended software because is the one that requries the least knowledge if you are not familiar with using a terminal or Linux. You can download the newest version here.

2. Our next step will be downloading the software that will help us with writing the image. My favourite one is Rufus so I’ll be using that. You will need to insert your SD card or USB into your PC and open up Rufus. Then, you will select the Device on which you want to write the image and SELECT the image that you want to write and then you press START.

Installation process with a monitor

After the proccess of writing the image is complete you will need to insert the SD card/USB into your Raspberry PI and connect your HDMI and power supply as well as your peripherals (keyboard and mouse). After a few seconds you should be greeted with a booting screen and a configuration process of your new OS in which you will select your country, timezone, language and the password of your wifi network. Now, If you connect your PI to WIFI, the system will ask you if you wish to automaticaly update your software packages to the newest version (If you don’t want to connect to WIFI now, or you want to use the ethernet cable, I will show you how to do so later on in this guide).

Installation process without a monitor

If you are like me and you had no idea that Raspberry PI 4 B comes with a mini HDMI port and not a normal sized version, this guide is for you!

The proccess is similar to an installation with a monitor, but first we need to add one more file into the booting section so Raspberry PI will know we want to connect to it over SSH. Now we need to create an empty file named “SSH”. The tricky part is that windows isn’t exactly the best thing for doing this easily. First, we need to right click>new text document and call it SSH. However, we just made a text file and we need an empty file right? To do this, we need to open our SSH.txt file and in the left upper corner click File>Save as and we will rename it to “SSH” with the quotation marks and select our Save as type as All Files. Then just press save and we should have an empty file called SSH.

After this you can insert the SD card/USB into a PI, connect all your peripherals (keyboard and mouse) and a network cable. The other side of the network cable connects to your PC.

We will need some sort of SSH client. One of the most popular and user friendly ones is the PuTTY. You can download it here. After you have downloaded and installed PuTTY, type raspberrypi.local into a hostname and select port 22. After that just select Open.

After that you will be prompted with the console asking for your login information. The default login and password information are

Login: pi

Password: raspberry

After this we need to enable the VNC interface which allows us to send display output over the network interface. We can do this by typing

 pi@raspberrypi:~ $ sudo raspi-config

You will see this menu in which you will select Interface Option and then P3 VNC.

Then go back into the menu again and select System Options and S5 Boot / Auto login and then B3 Desktop.

And now just leave the menu with ESC and for a reboot type 

sudo shutdown -r +0

After a few minutes your PI should be up again and we can continue.

We will need to download the VNC that serves as a virtual bridge to our PI display interface. Install and open the VNC and on the top side of the software type the hostname of our device which we previosly entered into the PuTTY is raspberrypi.local. Enter our username which is PI and password raspberry and press ok.

With that your setup of the Raspberry PI is completed and you can fully enjoy it as you would be able to with the connected display.

Final configuration of the Raspbian OS

After booting into your OS with or without a monitor you will need to make some finnal configurations of the system, such as updating your OS, changing your username, password, resolution etc. I will make another guide for this soon, but for now, have a nice day and enjoy your piece of PI.
